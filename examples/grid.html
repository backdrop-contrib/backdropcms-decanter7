.grid-* Helper Classes

The .grid-2, .grid-3, .grid-4 and .grid-6 helper classes can be used with the .rs-grid class to alter the default width of columns within the grid.

For example: if you add .grid-4 to a grid, then each child will span 3 columns by default laying out into a 4 column grid.

Note: The helper classes do not change the underlying structure of the 12 column grid. Tailwind column span classes should still work and will still be based on a 12 column grid. For example: to span half of a .grid-4 grid you would use .col-span-6 not .col-span-2.

There are some helper aliases to make this less confusing: .col-half, .col-quarter, .col-third, .col-twothird, .col-threequarter, .col-full.
